    docker与虚拟机的区别 #

    1. 物理机
      类比：别墅，独立花园
    2. 虚拟机
      类比：小区套房，共享花园，独立卫生间
    3. docker
      类比：胶囊房，共享花园，共享卫生间，独立床位

    VM实现是使用虚拟化引擎虚拟硬件层；运行程序需要安装完整镜像的操作系统；

    docker实现是居于操作系统之下；

    linux为例：使用linux下namespace与cgroup方式实现,
